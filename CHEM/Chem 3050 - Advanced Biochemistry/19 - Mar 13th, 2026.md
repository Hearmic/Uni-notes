#### Genome editing from double-stranded breaks
- Cytidine deaminases catalyze the deamination of C to U. which base pairs as T
- Most known cytidine deaminase enzymes operate on RNA
- can also act on single-stranded DNA
- Fusion of a cytidine deaminase (APOBEC1) with deactivated Cas9 (dCas9)
- ssDNA available for base editing

#### PAM (Protospacer Adjacent Motif) is a short, specific DNA sequence (usually 2–6 base pairs) immediately following the target DNA sequence, essential for CRISPR-Cas9 to bind and cut.

Base editing: key details
- 1. Target C must be present within position 4-8 (~15 bases upstream of the PAM site)
- 2. Base editing is sensitive to the preceding base (context) $T\underline{C} > A\underline{C}\ or\ C\underline{C} >> G\underline{C}$ 
- 3. Base editing is processive (multiple substrates will be modified, if present)
